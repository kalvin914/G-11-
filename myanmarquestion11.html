၁။(က) အောက်ပါတို့ကို မှန်လျှင်(မှန်) မှားလျှင်(မှား) ဟုရေးပါ။ (၁၀) မှတ်
(၁) ဆရာမငွေတာရီ၏အမည်ရင်းမှာ ဒေါ်ခင်ရီဖြစ်သည်။
(၂) ကျွန်ုပ်တို့၏နိုင်ငံတော်ဆောင်းပါးကို ပျဉ်မနားမောင်ချိုသင်း ရေးသားခဲ့သည်။
(၃) ရုံးစာအမျိုးမျိုးမှာ အမှာစာ ၊ အစီရင်ခံစာ ၊ အမိန့်စာ တို့ဖြစ်သည်။
(၄) မြန်မာစာရေးသားရာတွင် ပြောဟန်တစ်နည်းသာရှိသည်။
(၅) ဆားဒယ်စကားပြေကို ကိုလိုနီခေတ်တွင်ရေးသားခဲ့သည်။
(၆) ဆရာဇော်ဂျီသည် ရန်ကုန်တက္ကသိုလ်က ဝိဇ္ဇာဘွဲ့ရရှိခဲ့သည်။
(၇) ဦးသိန်းဟန်၏ ဇာတိမှာ မြောင်းမြမြို့ဖြစ်သည်။
(၈) သင်သေသွားသော် ကဗျာကို ဆရာဇော်ဂျီရေးသားခဲ့သည်။
(၉) ဆရာမင်းသုဝဏ်သည် ၁၉၆၃ခုနှစ်တွင် သီရိပျံချီဘွဲ့ရရှိခဲ့သည်။
(၁၀) အလှရှိရာကဗျာသည် လင်္ကာကဗျာအမျိုးအစား ဖြစ်သည်။
၁။(ခ) အောက်ပါတို့ကို ကွက်လပ်ဖြည့်ပါ။ (၁၀) မှတ်
(၁) ရေးဟန်မှာအပြောမဆန်ဘဲ_______ရေးသားဟန်မျိုးဖြစ်သည်။
(၂) မြန်မာစကားပုံတစ်ခုမှာ_______မြှားကုန်။
(၃) ပြောဟန်နှင့်ရေးဟန်တွင် အဓိကခြားနားသည်မှာ_________များဖြစ်သည်။
(၄) မဲဇာတောင်ခြေကဗျာသည်________နှင့်________ညီသည်။
(၅) ဆရာမင်းသုဝဏ်၏ အမည်ရင်းမှာ________ဖြစ်သည်။
(၆) ၁၉၆၄ခုနှစ်တွင်________ကဗျာပြိုင်ပွဲတွင်"မောင်ကြီးတို့ရွာ"ကဗျာဖြင့်ပထမဆုရရှိခဲ့သည်။
(၇) ဂျိမ်းဖော့ပွဲတော်ကဗျာသည်________ကဗျာအမျိုးအစားဖြစ်သည်။
(၈) ဦးမြင့်ဆွေသည် မိုးသောင်ယံကဗျာများ ၊ အလင်းခေတ်ကဗျာများ ၊ _________ကဗျာများမှာ ထင်ရှားသည်။
(၉) ငွေတာရီ၏ ဖခင်သည်_______ရုပ်ရှင်သူဌေးဦးညွှန့်ဖြစ်သည်။
(၁၀) ရတုသုံးပိုဒ်ဖွဲ့လျှင်_______ဟုခေါ်သည်။
၁။(ဂ)‌ အောက်ပါတို့မှ အဖြေမှန်ကိုရွေးပါ။ (၁၀) မှတ်
(၁) အသံကြိုးတုန်၍ ရွတ်ဆိုရသော ဗျည်းသံကို_______ဟုခေါ်သည်။
( သံတင်း ၊ သံပြင်း ၊ သံလေး )
(၂) အင် ၊ အန် ၊ အွန်သရတို့သည်_______သရတို့ဖြစ်သည်။
( ရုပ်သံသရ ၊ နှုတ်ခမ်းဝိုင်းသရ ၊ နှာသံပါသရ )
(၃) ပီမိုးနင်း၏အလုပ်ဆောင်းပါးသည်______စာအုပ်မှရွေးချယ်ချက်ဖြစ်သည်။
( အောင်မြင်ရေး ၊ တိုးတက်ရေး ၊ ထူးချွန်ရေး )
(၄) ဆရာမင်းသုဝဏ်သည် ၁၉၅၀ပြည့်နှစ်တွင်______ဘွဲ့ရခဲ့သည်။
( ဝဏ္ဏကျော်ထင် ၊ သီရိပျံချီ ၊ မဟာဝိဇ္ဇာဘွဲ့ )
(၅) ဆရာမငွေတာရီသည် အလှရှိရာကဗျာကို _________ နိုင်ငံမှပြန်ရောက်ပြီးနောက်ရေးသားခဲ့သည်။
( ပြင်သစ် ၊ ဂျပန် ၊ အ...)
(၆) ဆရာမျိုးမြင့်ဆွေသည် _______ ပညာဖြင့်လည်း အနုပညာအသက်မွေးဝမ်းကျောင်းပြုခဲ့သူဖြစ်သည်။
( ပန်းချီ ၊ ဂီတ ၊ စာပေ )
